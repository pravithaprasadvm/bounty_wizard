## Project Overview
Features:
- 3-step bounty wizard (Basics → Rewards → Backer)
- Sidebar + TopBar with page number
- Validation and state persistence via context
- Preview and Result page
- Simulated submit using sessionStorage

## How to run
1. npm install
2. npm run dev

## Notes
- Color #0085FF used for nav and primary buttons
- TopBar shows current page number
